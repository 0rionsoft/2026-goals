# 2026 Personal Goals + UK Time-Off Plan

## Personal goals for 2026 (pick 6–8 and track them)

### Health / energy
- **Train 3x/week for 40+ weeks** (misses allowed; consistency > perfection).
- **10k steps/day average** (or **70k/week**).
- **Sleep:** average **7h 15m+** on weeknights (track for 3 months, then adjust).
- **One repeatable “baseline” meal** you can eat **3–4x/week** to keep nutrition easy.

### Build / skills (dev + stream friendly)
- **Ship 2 portfolio projects** (Next.js + Supabase works great):
  - Project 1 live by **end of April**
  - Project 2 live by **end of September**
- **Publish weekly progress** (stream/YouTube/notes): **40 updates in 2026**.
- **Automate one annoying thing per month** (Discord/OBS/dev workflow): **12 small wins**.

### Money / stability
- **Emergency fund:** build to **3 months** expenses (or auto-transfer **+£___/month**).
- **Big purchase rule:** anything over **£___** needs **48 hours** + written reason.

### Relationships / life
- **48 “proper hangs”**: date night / mate night / family catch-up (~1/week).
- **Two trips booked early:** one spring/summer, one autumn/winter.

### Fun
